## Overview

A Z80 single-board computer / game console, including hardware schematics, software, and an emulator.

## Features

* Z80 CPU @ 8 MHz
* 8 KiB ROM, 8 KiB RAM
* 2 controller input ports
* 1-bit bitmap graphics (256x256 resolution, 8 KiB VRAM)
* PAL video output
